# Lavender Haze - Tienda de Vinilos 🎵

Este proyecto es una página web diseñada para una tienda de vinilos. El objetivo principal es mostrar un catálogo de vinilos, reseñas de clientes y un formulario de contacto, aplicando conceptos fundamentales de **HTML5** y **CSS3**.

## (Propósito)
La página fue creada para ofrecer una experiencia visual "retro-moderna", facilitando a los usuarios la exploración de álbumes clásicos y modernos con un diseño responsivo que se adapta a computadoras y dispositivos móviles.

## Características Principales 🚀
- **Diseño Retro:** Estética basada en colores crema, naranja y turquesa con bordes gruesos y sombras sólidas.
- **Catálogo de Productos:** Cards detalladas con imágenes de portadas, descripción y precios alineados para cada vinilo.
- **Menú Mobile:** Sistema de navegación funcional para celulares mediante checkbox.
- **Sección de Reseñas:** Espacio dedicado a la opinión de los usuarios con un diseño de cuadrícula flexible.
- **Banner Interactivo:** Sección principal con introducción, botones de acción y un video insertado de YouTube.

## Tecnologías Utilizadas 🛠️
- **HTML5:** Estructura semántica (header, main, section, footer).
- **CSS3:** - Flexbox para alineación y distribución de elementos.
  - Media Queries para diseño responsivo.
  - Posicionamiento absoluto para etiquetas de novedades.

## Estructura de Archivos 📁
- `index.html`: Estructura principal de la página.
- `styles.css`: Estilos, fuentes y diseño responsivo.
- `/img`: Carpeta que contiene las portadas de los vinilos y el logo.
